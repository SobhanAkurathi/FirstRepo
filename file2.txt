Version Control
---
-> It is a system that records all changes and modifications to files for tracking purpose
-> Developer use source control or source code management
-> It keeps track of changes when we make changes and who make the changes by specifying time and date and user
-> Developers use this for peer review 
-> There are different types of changes, such as adding new files, modifying or updating files, and deleting files

Continuous Integration(CI)
--
-> Used to automate the integration of code from multiple developers into a single main stream
Continuous Delivery:
Once code merged get merged into main stream it prepares for deployment On
Continuous Deployment is an extension of Continuous Delivery. 
The goal of Continuous Deployment is to deploy and release software to customers frequently and safely.
